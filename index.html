Amazon Clone

A fully functional Amazon clone built with ReactJS. This project emulates some of Amazon's features and functionalities.

Difficulties: During the development, I encountered a challenge with the shopping cart functionality. Specifically, I found that it was not updating correctly when a user added or removed items from their cart. After some investigation, I realized that the issue stemmed from how the data was being managed and passed between different components in the application.

Solution: To address this problem, I had to restructure the data flow and implement a more robust state management system using React Redux.

Live Code Editor

The Live Code Editor is a sandbox environment for users to create and test HTML, CSS, and JavaScript code snippets.

Difficulties: One of the major challenges for this project was creating the code preview feature and making sure it worked seamlessly.

Solution: To overcome this challenge, I had to dive into ReactJS's lifecycle methods to update the preview area in real-time. This required a lot of research and experimentation, but ultimately, I was able to achieve a solution that resulted in a very smooth experience.

The end result was an user-friendly code preview feature that allows users to showcase, test, and refine their HTML, CSS and JavaScript code.
