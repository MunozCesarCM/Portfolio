Amazon Clone

A fully functional Amazon clone built with ReactJS. This project emulates some of Amazon's features and functionalities.

Difficuties: During the development, I encountered a challenge with the shopping cart functionality. Specifically, I found that it was not updating correctly when a user added or removed items from their cart. After some investigation, I realized that the issue stemmed from how the data was being managed and passed between different components in the application.

Solution: To address this problem, I had to restructure the data flow and implement a more robust state management system using React Redux.
